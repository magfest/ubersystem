API Reference

The API is based on JSON-RPC, and conforms to the JSON-RPC spec. API calls should be made using HTTP POST requests. The request body should be a JSON formatted string including the following properties:

method
service and method name in dotted format "service.method"
params
method parameters, if relevant, in an array

The response is returned as a JSON formatted object. A successful response will have a "result" property. An error response will have an error message in an "error" property.

The API endpoint URL is:
{{ c.URL_BASE }}/jsonrpc/ 

{
  "method": "attendee.search",
  "params": ["{{ admin_account.attendee.full_name }}"]
}

{
  "id": null,
  "jsonrpc": "2.0",
  "result": [
    {
      "first_name": "{{ admin_account.attendee.first_name }}",
      "last_name": "{{ admin_account.attendee.last_name }}",
      "email": "{{ admin_account.attendee.email }}",
      "id": "{{ admin_account.attendee.id }}"
    }
  ]
}

{
  "id": null,
  "jsonrpc": "2.0",
  "error": {
    "code": -32603,
    "message": "Unexpected error"
  }
}

Authentication

Third party applications can authenticate requests by adding the X-Auth-Token header to each HTTP POST. You can find your auth tokens, or generate a new token, on the API Access page.

Requests without an X-Auth-Token header will return 401 Unauthorized. Requests with an invalid X-Auth-Token header will return 403 Forbidden. 

X-Auth-Token: XXXXXXXX-XXXX-XXXX-XXXX-XXXXXXXXXXXX

Command Line Example

The API can be tested by making requests from the command line using curl. In the given example, replace "XXXXXXXX-XXXX-XXXX-XXXX-XXXXXXXXXXXX" with a token from the API Access page. 

curl -v -k -d '{"method": "config.info"}' -H "X-Auth-Token: XXXXXXXX-XXXX-XXXX-XXXX-XXXXXXXXXXXX" -w "\n" {{ c.URL_BASE }}/jsonrpc/

Note on 404 Not Found

Methods that are expected to return a single specific object, like attendee.lookup, will raise a 404 Not Found if the object is not found.

Methods that are expected to return many objects, like attendee.search, will return 200 OK with an empty array if no matching objects are found.
